PersonaScope · 用户心智模型可视化分析面板
一个交互式数据分析面板，可根据用户行为轨迹，动态生成心理侧写雷达图与策略性商品推荐。项目包含 6 个内置典型用户画像，实现了心理状态与可视化图表的实时联动。

功能特性
6 个内置用户案例（深夜程序员、学生党、自由职业者等），点击左上角直接切换

心理侧写雷达图（行为特征、理性维度、社交活跃度等）动态刷新

状态标签（潜伏模式、防御卸载、沉浸逃避）与雷达图高亮联动

行为轨迹时间轴（无限制滚动）

策略性推荐内容与商品（零食大礼包、Steam 游戏、数码手办等）

深色 / 浅色模式一键切换

技术栈
前端：原生 HTML5 / CSS3 / JavaScript (ES6)

图表可视化：ECharts

AI 分析（可选）：DeepSeek API（演示模式下使用内置静态数据，无需 API Key）

样式方案：CSS 变量实现深色/浅色模式无缝切换

部署：GitHub Pages

项目亮点
6 个预设用户画像：涵盖多种用户类型，无需依赖外部 API 即可完整体验

状态驱动 UI 联动：切换案例时雷达图、推荐内容、时间轴同步刷新

全场景适配：支持深色/浅色模式，响应式布局适配不同屏幕尺寸

数据可视化：雷达图直观展示用户多维度心理侧写

真实场景还原：每个案例包含完整时间轨迹与消费偏好，展示行为分析完整链路
