📖 项目：User Profiler | 用户行为状态侧写系统

“不再停留于“用户是谁”，而是理解“用户此刻在关注什么。”

🎯 核心理念：人是情感与状态的集合
传统用户画像依赖年龄、性别、消费能力等静态标签，但真实的人并不是固定不变的。
一个日常消费都要货比三家的人，可能会为了热爱的兴趣毫不犹豫地掷下千金。
同一个人，在微信上平和克制，在微博、抖音等平台里放肆宣泄。
因为消费意愿永远与情感投入深度绑定，而非收入。

User Profiler 希望解决的，不是“给用户贴标签”，而是更准确地理解用户当前的心理状态与真实需求。
通过用户在不同时间、不同平台的行为碎片，识别其即时心理状态，找到驱动真实行为的情感锚点，从而给出更精准的触达策略。

⚡ 核心优势（与传统画像的本质区别）

维度 传统用户画像 侧写系统 (User Profiler)
视角 静态标签（假设人是固定不变的） 动态状态（承认人的复杂与跳跃）
核心逻辑 25岁女性 → 机械地推送美妆 识别“主动成长模式” → 精准推送学习与效率工具
数据维度 人口统计学 时间× 平台× 动作× 情感动向

🛠️ 产品如何工作？
你只需要输入用户在不同时间、不同平台的行为碎片，系统将通过特有的情感锚点算法，像心理侧写师一样，为你拼凑出最真实的动态档案：
[捕捉行为] ──> [识别即时状态] ──> [跨平台交叉对比] ──> [定位情感锚点] ──> [输出推送策略]

📋 侧写档案核心内容：
🙂 今日状态分布图：实时呈现用户在不同时段的心理状态，包括焦虑、专注、寻求补偿、情感宣泄等维度分布。
🎯 情感锚点判定： 剥离表面行为，直击用户当前最核心的动机（如：自我提升、孤独慰藉、压力释放）。
🚀 动态推送策略建议： 不再盲目群发，给出行之有效的即时触达方案。

🎇 产品定位
User Profiler 不是又一个数据看板。
它更像一份随时可以翻阅的用户备忘录——记录的不是"这个人是谁"，而是"这个人现在在意什么"。
让每一个数据，都变成有温度、有呼吸的故事。
